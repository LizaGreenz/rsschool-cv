# Code examples
Complete the square sum function so that it squares each number passed into it and then sums the results together.
```
function squareSum(numbers)
{ 
  var totalSum = 0;
  for (i=0;i<numbers.length;i++)
  {
    totalSum += Math.pow(numbers[i], 2);
  }
  return totalSum;
}
```
